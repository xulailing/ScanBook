## 扫扫图书

官网：[http://fangjie.info/scanbook](http://fangjie.info/scanbook)

> * 书，是人类的灵魂。
> * 扫扫图书是一个让你懂得如何去选择一本书的APP。
> * 这里你可以扫描条形码查询图书，
> * 你也可以关键字搜索，遇到合乎你口味的书，
> * 你还可以看看别人的读书笔记，不同角度去体会。
